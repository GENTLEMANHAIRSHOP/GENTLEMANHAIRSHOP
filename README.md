# Hi, I'm Seryun Hong 👋

I'm a student developer interested in AI Vision, Embedded Systems, IoT, Physical Computing, and future technology.

I enjoy building projects that connect software with real-world hardware, such as AI cameras, Arduino controllers, Raspberry Pi systems, and interactive game devices.

## Featured Projects

### 🐕 AI Lost Pet Detector
AI camera-based system that detects lost pets or stray dogs and sends real-time Discord alerts.

- Raspberry Pi / webcam / HuskyLens concept
- Discord Webhook alert system
- Expandable to YOLO-based object detection
- Local community safety project

### 🥊 Arduino Fighting Game Booth
Interactive school festival booth that turns punch and kick actions into keyboard inputs for a fighting game.

- Arduino Leonardo Keyboard HID
- Pressure sensors inside dolls or cushions
- Physical computing game controller
- Expandable with ESP32-CAM motion recognition

### 🌺 Mugunghwa Physical Board Game
Arduino-based physical computing board game inspired by "무궁화 꽃이 피었습니다".

- Arduino Uno game logic
- Servo motor player movement
- LED and buzzer feedback
- Movement detection and warning system

### 🔊 Korean AI Smart Speaker
Raspberry Pi-based Korean AI smart speaker concept using a hotword, AI response generation, TTS, and LCD facial expressions.

- Korean hotword: "블스"
- Python-based conversation flow
- ChatGPT API expansion plan
- LCD expression display concept

### ⚽ AI Autonomous Sports Referee
Future-job idea project that demonstrates a webcam-based AI referee system.

- Laptop webcam input
- OpenCV-based ball tracking
- Goal line and out line decision demo
- Expandable to YOLO, pose estimation, and rule engines
